## 1. 初期データ

PCは以下のデータを初期から持つ。

| 項目         | 値                                      |
| ------------ | --------------------------------------- |
| HP           | 100                                     |
| スタン最大値 | 5                                       |
| マナ所持上限 | 火・水・地・風 各5、光・闇 各1（詳細は `alchemy_and_items.md`） |

---

## 2. キャラクター作成の手順

キャラクター作成は、以下の順に行う。

1. 作成できるアイテム種を1つ選ぶ
2. 種族を1つ選ぶ
3. パーソナルデータを決める

---

## 3. アイテム種の選択

以下のアイテム種の中から1つを選択する。  
アイテム種を選ぶと、自動で付き従う精霊も決定される。

| アイテム種 | 精霊 | 担当役割                         |
| ---------- | ---- | -------------------------------- |
| 薬         | 水   | HP回復、デバフ解除など           |
| 爆弾       | 火   | 相手へのダメージ                 |
| 香料       | 土   | 味方への強化                     |
| 札         | 風   | 敵への妨害、デバフ               |

選んだアイテム種のレシピを錬金・想成で使用できる。扱えない系統のレシピは使用不可とする。

種族の戦闘時能力（付喪の炉火純青など）が参照する「キャラ作時に選択したアイテム種」も、この選択を指す。

---

## 4. 種族の選択

6つの種族（人間・獣裔・機巧・錬生・付喪・半霊）のうち1つを選ぶ。  
各種族の能力詳細は `races.md` を参照する。

---

## 5. パーソナルデータ

最低限、名前と職業を決める。あとは自由に決めてよい。

「職業」は町での顔・雰囲気の設定であり、機械的な制約とは独立する。

---

## 6. 初期行動値

| 項目       | 内容                                           |
| ---------- | ---------------------------------------------- |
| 決定方法   | 戦闘開始時に `1d10` を振り `floor(n/2)` で決定 |
| 結果範囲   | 0〜5                                           |
| タイミング | 戦闘開始前に1回だけ振る                        |

行動値の詳細な扱い・タイムラインへの配置は `combat.md` を参照する。

---

## 7. 1日の製作数

`day_cycle.md` の街務ルールに準拠する。  
昼フェイズ2回・夕フェイズ2回の**最大4回/日**が錬金の上限となる。
