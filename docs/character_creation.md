## 1. キャラクター作成の概要

キャラクターは以下の要素から構成される。

- **系統**：扱えるレシピの種類を決める機械的専門性
- **町での顔**：職業・商い・雰囲気の自由設定。機械的な制約とは独立する
- **種族**：判定時のダイス能力を決める（→ `races.md`）
- **基本ステータス**：HP・初期行動値

キャラクター作成時に行うことは以下の順序になる。

1. 系統を選ぶ
2. 種族を選ぶ
3. 町での顔を決める
4. 基本ステータスを確認する

> キャラクターシートのレイアウトはテストプレイ後に整備する。

---

## 2. 系統選択（テストプレイ版）

テストプレイ段階では、以下の4クラス固定から**1つ**を選ぶ。  
系統の自由選択は正式ルールで実装する。

| クラス        | 系統 | 担当役割       |
| ------------- | ---- | -------------- |
| 火力          | 爆弾 | 単発ダメージ   |
| 回復          | 薬   | 単発回復・治療 |
| バフ          | 香料 | 単発強化       |
| デバフ/スタン | 札   | 単発妨害・封印 |

---

## 3. 系統選択（正式ルール）（仮）

正式ルールでは、アイテム種の系統を**2種**自由に選ぶ。  
選択可能な系統は `alchemy_and_items.md` に定義する消耗品・装備カテゴリに準じる。

- 選んだ2系統のレシピを使用できる
- 工房の成長に連動して、習得系統を増やすことができる

> 正式ルールの系統一覧・成長連動ルールは後で整備する。

---

## 4. 未習得系統のルール

- 未習得の系統に属するレシピは**使用不可**とする
- 未習得系統の素材をダイスプールに追加することはできない
- ペナルティ付き使用（難易度上昇など）ではなく、完全に使用不可として扱う

## 5. 基本ステータス

### HP（仮）

| 項目   | 値                                                                            |
| ------ | ----------------------------------------------------------------------------- |
| 初期HP | **100**                                                                       |
| 補足   | 達成値基準（1d10 + マナ2〜3個の期待値 ≒ 17〜22）の約5倍。平均4〜5撃耐久を想定 |

- HP回復タイミングは `day_cycle.md` に準じる（HP全快は日単位）
- 防御値は装備品の固定値である（将来的には武器と同様に品質から算出する設計を整備する）

### スタン最大値

| 項目         | 値                                                                                     |
| ------------ | -------------------------------------------------------------------------------------- |
| スタン最大値 | **5**                                                                                  |
| 補足         | スタン値がこの値以上になると手札を1枚失い、ノックバック発生。詳細は `combat.md` を参照 |

### 初期行動値

| 項目       | 内容                                           |
| ---------- | ---------------------------------------------- |
| 決定方法   | 戦闘開始時に `1d10` を振り `floor(n/2)` で決定 |
| 結果範囲   | 0〜5（0と5が出にくい分布）                     |
| タイミング | 戦闘開始前に1回だけ振る                        |

- 行動値の大きいキャラクターほど最初にタイムライン後方に配置される
- 行動値の詳細な扱い・タイムラインへの配置は `combat.md` を参照

## 6. マナ所持上限

各属性マナの個人所持上限は以下のとおり。詳細は `alchemy_and_items.md` を参照。

| 属性   | 対応ダイス | 所持上限 |
| ------ | ---------- | -------- |
| `<火>` | d4         | 5        |
| `<地>` | d6         | 5        |
| `<風>` | d8         | 5        |
| `<水>` | d20        | 5        |
| `<闇>` | d10        | 1        |
| `<光>` | d12        | 1        |
