## 想成のスコーリア

作品名は **「想成のスコーリア」** です。

「スコーリア」は scholia / scholion を下敷きにした語感で、注釈・解釈の痕跡・積み重なる読みを表します。

**想成** は、形を思い浮かべ、いったん成立した像を呼び戻し、解釈によって現実へ結び直す行為です。

---

## 霧域

町の外には **霧域** と呼ばれる未開地があります。

- 同じ場所を目指しても毎回道筋が違う
- 地図に描いても現地で噛み合わない
- 見通しが悪く、景色が定まりにくい

繰り返し探索を行うと、**スコーリオン** と呼ばれる遺物が時折見つかります。

---

## スコーリオンと地景

作中人間にとってスコーリオンは、 **霧域の土地を開くための界標** です。

スコーリオンを据えることで霧が晴れ、 **地景** が拓かれます（[マップと地景](map.md)）。

地景想成は、理屈の完全には分かりきっていない **術・技法** として理解されています。

## 四属性と光・闇

### 火・地・風・水

| 属性 | 意味の例                                       |
| ---- | ---------------------------------------------- |
| 火   | 熱、変化、活性、加工、勢い                       |
| 地   | 固定、基盤、重さ、保持、安定                     |
| 風   | 流動、操作、拡散、機転、移動                     |
| 水   | 変容、浸透、柔軟、循環、曖昧さ                   |

### 闇・光

- **闇** — 解釈・創造・可能性・未定形性に近い
- **光** — ただ在るもの・解釈される前の在り方に近い

## 精霊

精霊は、火・地・風・水の働きに近い存在です。

[キャラクター作成](character.md) で選んだアイテム種に応じて、付き従う精霊が決まります。

## マナ

マナとは、草木・鉱石・獣・遺物などに宿る力です。

砕いたり、精製したり、術式によって整えることで取り出し、錬成や想成の燃料として使います。

人間の内から無尽蔵に湧くものではなく、 **物から取り出して扱う力** という理解が基本です。

詳細は [錬金とアイテム](alchemy.md) を参照してください。
