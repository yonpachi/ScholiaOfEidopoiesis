# day_cycle.md

最終更新: 2026-06-22  
担当範囲: 日内の進行順、行動できるタイミング、持続や持ち越しの共通処理  
このファイルに書かないもの: 個別印記の定義（→ `marks_and_tags.md`）、攻撃やスタンの戦闘処理詳細（→ `combat.md`）、探索カード処理本文（→ `exploration.md`）

## 1. 一日の構造

本作の一日は、次の順で進行する。

**朝フェイズ → 昼フェイズ → 夕フェイズ → 夜フェイズ**

一日ごとに準備を整え、探索に出て、素材を持ち帰り、アイテムを作り、納品していくのが基本的な流れである。  
この一日そのものが、実質的なプレイ単位である。

## 2. 朝フェイズ

朝は、その日の準備や開始処理を行う時間である。以下の順に沿って処理する。

1. 日付が進む（曜日変化の影響を受ける要素があれば更新する）
2. HPの回復（各PCはHPを最大値まで回復する）
3. 評判の自然低下
4. 依頼の期限日数の減少（残り0日になった依頼は、その日の夜が最後の納品機会）
5. 依頼の追加（その日に受けられる新しい副依頼を追加する。主依頼が新しく提示される場合もこのタイミング）
6. 依頼の受注
7. スコーリオンの配置（手に入れている場合、界標として据える）
8. 施設の設置（研究が完了した施設などを、素材やお金を払って設置する）

### HP回復の例外

- **全滅から帰還したPCは HP1 で帰還し、その日は回復しない**

## 3. 昼フェイズ

昼は自由に行動できる時間である。

まず霧域や地景に**探索**に出るか、街の中で**街務**を行うかのどちらか1つだけを選択する。  
PC全員が同じ選択をする必要はなく、別行動も可能である。

### 探索

探索を選んだPC全員で霧域か地景1つを選択し、その場所の探索を行う。  
探索中は街にいる人や倉庫とのやり取りは原則できない。  
この時間の終了時、探索を行ったPCは街に帰還する。  
持ち帰った素材は倉庫へ入るが、そのフェイズ中に追加で街の行動を行うことはできない。

詳細は `exploration.md` を参照する。

### 街務

街にいる間は、相談や演出、確認、倉庫とのやり取りなどは自由に行える。  
ただし、ルール上の効果を持つ行動は街務回数を消費する。

街務を選んだPCは、以下の行動を**2回**まで選んで行うことができる。同じ行動を2回選んでもかまわない。

1. **錬金** — PCの作成できるアイテムから1つを選び、錬金を行う
2. **買い物** — 現在この街に存在しているお店から自由に買い物を行う
3. **交流** — 街の住人や街にいる他PCと交流を深める
4. **調査・研究** — 現在の依頼のために必要な調査や、街・工房を発展させるために必要な研究を行う

## 4. 夕フェイズ

夕も昼と同様、自由に行動ができる時間である。探索か街務のどちらか1つを選択し行う。  
街務を選んだPCは、昼と同じく2回まで行動を行える。

ただし夕に探索を行う場合、その探索には**危険度+2**の補正がかかる。

## 5. 夜フェイズ

夜は、その日の成果を確定する時間である。以下の順に沿って処理する。

1. 依頼の達成（受注している主依頼や副依頼の達成条件を満たしているなら、納品・確認を行う。期限が0日の依頼が達成できなかった場合、ここで失敗）
2. 評判の反映（依頼などの成否に応じて、現在の評判を増減させる）
3. 研究成果の確定（その日に進めた研究がある場合は、ここで成果を確定する。研究が完了した場合、翌日から施設の設置が行える）
4. その日の成果の確定（その日に得た素材や倉庫状況などを確認し、1日を終える）

## 6. 日内の共通タイミング

### 探索に出た後

- 探索は1フェイズ分を丸ごと消費する
- 昼フェイズ開始時、夕フェイズ開始時に、そのフェイズで探索に出るかを決める
- 探索に出ると決めた時点で、そのフェイズ中は町行動を行えない
- 探索から帰還後も、そのフェイズ中は町行動を行えない
- 倉庫との交換や町施設利用も止まる

### アイテム使用タイミング

- 町中: いつでも使用可能
- 探索中: 山札からカード2枚を公開する前まで
- 戦闘中: 各自の手番に1回（`combat.md` を参照）

## 7. 状態と持続

### HP0

- 探索中にHP0以下で帰還したPCは、町へ**HP1で帰還**する
- 全滅時の処理は `combat.md` および `exploration.md` を参照する

### マナ

- マナは**完全持ち越し**とする

### バフ・デバフ

- バフやデバフは、基本的に**1戦闘中**または**1フェイズ中**のみ持続する
- それを超える持続は個別の効果文に書く

## 8. 倉庫と持ち帰り

- 探索中以外は倉庫との交換が可能
- 帰還した瞬間、素材は倉庫へ入る
- 手持ちアイテムはそのまま保持してよい

## 9. 全滅後の日程進行

- 昼フェイズに全滅した場合、その後は**夕フェイズ**へ移行する
- 夕フェイズに全滅した場合、そのまま**夜フェイズ**へ移行する
- 依頼期限や日程進行は通常通り処理する
