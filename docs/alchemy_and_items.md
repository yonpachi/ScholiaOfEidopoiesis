## 1. 基本単位

本作では、アイテムが基本単位である。

- タグはアイテムに付属する情報である
- PCはタグ単体を保持しない
- 素材を使う、分解する、持ち帰る、という実物ベースの処理を重視する

### アイテム種別

| 種別           | 概要                                       |
| -------------- | ------------------------------------------ |
| 素材           | 探索で得る。錬金材料・分解対象             |
| 消耗品         | 使うとなくなる（薬・爆弾・香料・札など）   |
| 装備品         | 武器・防具など。装備している時だけ効果発動 |
| 納品物         | 依頼達成用                                 |
| 設置物         | **未実装**                                 |
| その他         | 特殊アイテム                               |

## 2. タグとマナ

- 素材にはタグがある。タグは `[タグ名]` で表記する
- タグそのものを単独で持つことはできない
- タグごとに対応するマナがある
- タグと印記の共通定義は `marks_and_tags.md` を参照する

### タグ重複

- 同名タグの重複は許容しない
- 重複したタグは、1つ以外を余計なタグとして扱う

## 3. マナの扱い

- マナは6属性: `<火>` / `<地>` / `<風>` / `<水>` / `<闇>` / `<光>`
- 基本的には**アイテムを砕いて得る**
- マナは**個人管理**である

### マナ所持上限

| 属性   | 対応ダイス | 所持上限 |
| ------ | ---------- | -------- |
| `<火>` | d4         | 5        |
| `<地>` | d6         | 5        |
| `<風>` | d8         | 5        |
| `<水>` | d20        | 5        |
| `<闇>` | d10        | 1        |
| `<光>` | d12        | 1        |

## 4. 精霊

キャラクター作成時に選んだアイテム種に応じて、付き従う精霊が決まる（`character_creation.md` 参照）。  
精霊は、判定のたびにその属性マナ1を追加できる任意効果を持つ方向で整理している。

## 5. 品質・等級・重量

### 品質

品質は、そのアイテムそのものの出来を表す数値である。値が大きいほど効果が高くなる。  
装備品ではダメージや防御値に、消耗品では効果量に関わる。

### 等級

等級は、そのアイテムの格や仕様段階を表す数値である。品質が「その個体の出来」なのに対し、等級は「その仕組みや設計の段階」を表す。  
**等級は小さいほど強い値**である。主に装備品に付与される。

### 重量

全てのアイテムには重量（軽量・中量・重量）が設定されている。戦闘中の待機値に影響する（`combat.md` 参照）。

| 重量 | 待機値補正 |
| ---- | ---------- |
| 軽量 | −1         |
| 中量 | ±0         |
| 重量 | +1         |

## 6. 分解

分解は、アイテムを砕いてマナに変える処理である。分解したアイテムは失われる。  
分解で得られるマナは、そのアイテムが持つタグや効果によって決まる。  
進行に関わる特殊なアイテムなどは分解できない場合がある。

戦闘中の分解は `combat.md` を参照する。

## 7. 錬金

錬金は、素材を組み合わせて新しいアイテムを作る行動である。  
錬金は街にいる間の**街務**として行う（`day_cycle.md` 参照）。

錬金で使用したレシピは保存され、使うたびに改良されていく。保存されたレシピは、以後**想成使用**で呼び出せる。

### 7-1. 錬金の流れ

1. レシピ選択
2. 材料選択
3. マナ準備
4. 判定
5. 結果処理

#### レシピ選択

使用するレシピを1つ選ぶ。PCは自分が扱える系統のレシピのみ使用できる。

#### 材料選択

レシピが要求するタグを満たすようにアイテムを選ぶ。素材だけでなく他のアイテムも材料にできる。  
使えるアイテムの数は、完成品に付与されているタグの数まで。  
完成品に必要ないタグが混ざる場合、それは**余剰タグ**として扱う。

#### マナ準備

レシピのマナコストで指定されたマナを支払う。不足していれば錬金は行えない。

#### 判定

材料を消費し、マナコストを支払ったら判定を行う。  
**1d10 + 支払った分のマナダイス**で判定する（`dice_and_checks.md` の手順に従う）。

- **難易度** = レシピに設定された難易度 + 余剰タグ1つにつき1
- 判定の結果が難易度以上なら成功、未満なら失敗
- 判定で算出した値を、今回の**錬金値**として扱う

#### 結果処理

使用したレシピに**熟練度**と**錬金値**を記録する。

- **熟練度**: そのレシピを使った回数
- **錬金値**: そのレシピに保存されている完成度
- **品質**: 熟練度と錬金値から決定する

**初めて使用するレシピの場合**（成功・失敗に関わらずレシピを保存する）:

| 結果 | 処理                                                                 |
| ---- | -------------------------------------------------------------------- |
| 成功 | 熟練度1、錬金値を今回の錬金値に設定。作成品の品質 = 熟練度 + 今回の錬金値 でアイテム1つ獲得 |
| 失敗 | 熟練度1、錬金値をレシピの難易度に設定。失敗作1つ獲得。材料・マナは戻らない |

**既に使用したことがあるレシピの場合**（成功・失敗に関わらず熟練度を1増やす）:

| 結果 | 処理                                                                 |
| ---- | -------------------------------------------------------------------- |
| 成功 | 今回の錬金値が保存値より高ければ錬金値を更新。作成品の品質 = 現在の熟練度 + 今回の錬金値 でアイテム1つ獲得 |
| 失敗 | 錬金値は更新しない。失敗作1つ獲得。材料・マナは戻らない |

### 7-2. 保存されたレシピと想成

保存されたレシピは想成使用で呼び出せる。  
**想成使用時の品質 = 保存されている熟練度 + 保存されている錬金値**

### 7-3. 失敗作

錬金に失敗した場合、失敗作を1つ獲得する。完成品としては扱わない。  
失敗作は、判定中にダイスロールを行った後・反応を行う前に使用できる。  
使用した場合、反応を行う前に、そのダイスロールを一度だけやり直せる。

## 8. 実体使用と想成使用

### 実体使用

手元にあるアイテムを実際に使うこと。消耗品を使った場合、そのアイテムは消費される。  
効果が打ち消されても戻らない。戦闘中の消耗品は基本的に実体使用として扱う。

### 想成使用

手元の実物を使わず、覚えているレシピをマナで再現して使うこと。  
指定されたマナを支払って効果を発動する。使用したマナは、効果が打ち消されても戻らない。  
戦闘中の想成は `combat.md` を参照する。

実体と想成は、同じアイテムを使用する場合でも効果が異なる場合がある。

## 9. 所持と倉庫

- 装備品や消耗品は、基本的にキャラクターが個人で所持する
- 素材は、素材カゴや倉庫で管理する
- 探索中に得た素材は素材カゴに入り、帰還後に倉庫へ移す

詳細は `economy_and_storage.md` を参照する。

## 10. 装備品

装備品は、キャラクターが装備して使うアイテムである。武器・防具など。  
**所持しているだけでは効果は発揮されない。**装備している時だけ有効である。

装備品は、品質・等級・重量・タグ・属性・耐性・効果を持つ。

### 10-1. 武器

武器は攻撃に使用する。武器属性（`<斬>` / `<打>` / `<突>`）と攻撃係数を持つ。マナ属性がある場合はそれも持つ。  
素手は `<打>` の武器として扱う。

**ダメージ = 品質 ÷ 等級 × 攻撃係数（端数切り捨て）**

攻撃係数と待機値は、行動段階と武器重量で決まる。表の左が待機値、右が攻撃係数である。

| 重量／行動段階 | 軽行動 | 中行動 | 重行動 | 特行動 |
| -------------- | ------ | ------ | ------ | ------ |
| 軽量           | 2 / 3  | 3 / 4  | 4 / 4  | 5 / 5  |
| 中量           | 3 / 3  | 4 / 6  | 5 / 6  | 6 / 7  |
| 重量           | 4 / 4  | 5 / 5  | 6 / 8  | 7 / 11 |

武器効果は攻撃効果適用の段階で処理する。特定の行動段階でのみ発動する効果もある。

### 10-2. 防具

防具は防御や回避に使用する。防御値、回避値、防御係数を持つ。

**防御値 = 品質 ÷ 等級（端数切り捨て）**

防御行動で得る `<装甲>` の強度:

**`<装甲>`の強度 = 品質 ÷ 等級 × 防御係数（端数切り捨て）**

防具重量ごとの回避値:

| 防具重量 | 軽量 | 中量 | 重量 |
| -------- | ---- | ---- | ---- |
| 回避値   | 4    | 3    | 2    |

防具重量表（左が待機値、右が防御係数）:

| 重量／行動段階 | 軽行動 | 中行動 | 重行動 | 特行動 |
| -------------- | ------ | ------ | ------ | ------ |
| 軽量           | 2 / 2  | 3 / 3  | 4 / 4  | 5 / 5  |
| 中量           | 3 / 3  | 4 / 4  | 5 / 5  | 6 / 7  |
| 重量           | 4 / 4  | 5 / 5  | 6 / 7  | 7 / 9  |

回避行動では、待機値だけを使用する。

### 10-3. 装備品と分解

装備品も分解できる場合がある。分解した装備品は失われる。得られるマナは装備品ごとの記述に従う。
