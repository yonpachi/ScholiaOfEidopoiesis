## マナ

マナは6属性 `<火>` / `<地>` / `<風>` / `<水>` / `<闇>` / `<光>` です。

基本的には **アイテムを砕いて得ます**。マナは **個人管理** です。

| 属性   | 対応ダイス | 所持上限 |
| ------ | ---------- | -------- |
| `<火>` | d4         | 5        |
| `<地>` | d6         | 5        |
| `<風>` | d8         | 5        |
| `<水>` | d20        | 5        |
| `<闇>` | d10        | 1        |
| `<光>` | d12        | 1        |

## 品質・等級・重量

### 品質

品質は、そのアイテムそのものの出来を表す数値です。値が大きいほど効果が高くなります。

### 等級

等級は、そのアイテムの格や仕様段階を表す数値です。**等級は小さいほど強い値** です。主に装備品に付与されます。

### 重量

全てのアイテムには重量（軽量・中量・重量）が設定されています。戦闘中の待機値に影響します（[戦闘](combat.md) 参照）。

| 重量 | 待機値補正 |
| ---- | ---------- |
| 軽量 | −1         |
| 中量 | ±0         |
| 重量 | +1         |

## 分解

分解は、アイテムを砕いてマナに変える処理です。分解したアイテムは失われます。

分解で得られるマナは、そのアイテムが持つタグや効果によって決まります。

進行に関わる特殊なアイテムなどは分解できない場合があります。

## 錬金

錬金は、素材を組み合わせて新しいアイテムを作る行動です。

錬金は街にいる間の **街務** として行います。

錬金で使用したレシピは保存され、使うたびに改良されていきます。保存されたレシピは、以後 **想成** で呼び出せます。

### 錬金の流れ

1. レシピ選択
2. 材料選択
3. マナ準備
4. 判定
5. 結果処理

#### レシピ選択

使用するレシピを1つ選びます。PCは自分が扱える系統のレシピのみ使用できます。

#### 材料選択

レシピが要求するタグを満たすようにアイテムを選びます。素材だけでなく他のアイテムも材料にできます。

使えるアイテムの数は、完成品に付与されているタグの数までです。

完成品に必要ないタグが混ざる場合、それは **余剰タグ** として扱います。

#### マナ準備

レシピのマナコストで指定されたマナを支払います。不足していれば錬金は行えません。

#### 判定

材料を消費し、マナコストを支払ったら判定を行います。

**1d10 ＋ 支払った分のマナダイス** で判定します（[判定](checks.md) の手順に従います）。

- **難易度** ＝ レシピに設定された難易度 ＋ 余剰タグ1つにつき1
- 判定の結果が難易度以上なら成功、未満なら失敗
- 判定で算出した値を、今回の **錬金値** として扱います

#### 結果処理

使用したレシピに **熟練度** と **錬金値** を記録します。

- **熟練度**: そのレシピを使った回数
- **錬金値**: そのレシピに保存されている完成度
- **品質**: 熟練度と錬金値から決定します

**初めて使用するレシピの場合**（成功・失敗に関わらずレシピを保存します）:

| 結果 | 処理                                                                 |
| ---- | -------------------------------------------------------------------- |
| 成功 | 熟練度1、錬金値を今回の錬金値に設定。作成品の品質 ＝ 熟練度 ＋ 今回の錬金値 でアイテム1つ獲得 |
| 失敗 | 熟練度1、錬金値をレシピの難易度に設定。失敗作1つ獲得。材料・マナは戻りません |

**既に使用したことがあるレシピの場合**（成功・失敗に関わらず熟練度を1増やします）:

| 結果 | 処理                                                                 |
| ---- | -------------------------------------------------------------------- |
| 成功 | 今回の錬金値が保存値より高ければ錬金値を更新。作成品の品質 ＝ 現在の熟練度 ＋ 今回の錬金値 でアイテム1つ獲得 |
| 失敗 | 錬金値は更新しません。失敗作1つ獲得。材料・マナは戻りません |

### 保存されたレシピと想成

保存されたレシピは想成で呼び出せます。

**想成使用時の品質 ＝ 保存されている熟練度 ＋ 保存されている錬金値**

### 失敗作

錬金に失敗した場合、失敗作を1つ獲得します。完成品としては扱いません。

失敗作は、判定中にダイスロールを行った後・反応を行う前に使用できます。

使用した場合、反応を行う前に、そのダイスロールを一度だけやり直せます。

## 実体使用と想成

### 実体使用

手元にあるアイテムを実際に使うことです。消耗品を使った場合、そのアイテムは消費されます。

効果が打ち消されても戻りません。戦闘中の消耗品は基本的に実体使用として扱います。

### 想成

手元の実物を使わず、覚えているレシピをマナで再現して使うことです。

指定されたマナを支払って効果を発動します。使用したマナは、効果が打ち消されても戻りません。

戦闘中の想成は [戦闘](combat.md) を参照してください。

## 装備品

装備品は、キャラクターが装備して使うアイテムです。武器・防具など。

**所持しているだけでは効果は発揮されません。** 装備している時だけ有効です。

### 武器

武器は攻撃に使用します。武器属性（`<斬>` / `<打>` / `<突>`）と攻撃係数を持ちます。マナ属性がある場合はそれも持ちます。

素手は `<打>` の武器として扱います。

**ダメージ ＝ 品質 ÷ 等級 × 攻撃係数（端数切り捨て）**

攻撃係数と待機値は、行動段階と武器重量で決まります。表の左が待機値、右が攻撃係数です。

| 重量／行動段階 | 軽行動 | 中行動 | 重行動 | 特行動 |
| -------------- | ------ | ------ | ------ | ------ |
| 軽量           | 2 / 3  | 3 / 4  | 4 / 4  | 5 / 5  |
| 中量           | 3 / 3  | 4 / 6  | 5 / 6  | 6 / 7  |
| 重量           | 4 / 4  | 5 / 5  | 6 / 8  | 7 / 11 |

### 防具

防具は防御や回避に使用します。

**防御値 ＝ 品質 ÷ 等級（端数切り捨て）**

防御行動で得る `<装甲>` の強度:

**`<装甲>` の強度 ＝ 品質 ÷ 等級 × 防御係数（端数切り捨て）**

| 防具重量 | 軽量 | 中量 | 重量 |
| -------- | ---- | ---- | ---- |
| 回避値   | 4    | 3    | 2    |

| 重量／行動段階 | 軽行動 | 中行動 | 重行動 | 特行動 |
| -------------- | ------ | ------ | ------ | ------ |
| 軽量           | 2 / 2  | 3 / 3  | 4 / 4  | 5 / 5  |
| 中量           | 3 / 3  | 4 / 4  | 5 / 5  | 6 / 7  |
| 重量           | 4 / 4  | 5 / 5  | 6 / 7  | 7 / 9  |
