## キャラクター作成の手順

キャラクター作成は、以下の順に行います。

1. 作成できるアイテム種を1つ選ぶ
2. 種族を1つ選ぶ
3. パーソナルデータを決める
4. 初期武器・防具を決める（[TODO](#todo)）

---

## 1. アイテム種の選択

以下のアイテム種の中から1つを選択します。  
アイテム種を選ぶと、自動で付き従う精霊も決定されます。

| アイテム種 | 精霊 | 担当役割               |
| ---------- | ---- | :--------------------- |
| 薬         | 水   | HP回復、デバフ解除など |
| 爆弾       | 火   | 相手へのダメージ       |
| 香料       | 土   | 味方への強化           |
| 札         | 風   | 敵への妨害、デバフ     |

選んだアイテム種のレシピを錬金・想成で使用できます。扱えない系統のレシピは使用不可とします。

種族の戦闘時能力（付喪の炉火純青など）が参照する「キャラ作時に選択したアイテム種」も、この選択を指します。

---

## 2. 種族の選択

6つの種族（人間・獣裔・機巧・錬生・付喪・半霊）のうち1つを選びます。

- 人間（ヒューム）
- 獣裔（ベスティア）
- 機巧（マキナ）
- 錬生（ホムンクルス）
- 付喪（レリキア）
- 半霊（ウンブラ）

各種族の能力詳細は [種族](ancestry.md) を参照してください。

---

## 3. パーソナルデータ

最低限、名前と職業を決めてください。あとは自由に決めて構いません。

「職業」は町での顔・雰囲気の設定であり、機械的な制約とは独立します。
