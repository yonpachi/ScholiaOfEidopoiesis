## 1. 基本ダイス

- 基本判定ダイスは **d10** とする
- 必要に応じて、属性マナに対応する**属性ダイス**を追加して振る

属性ダイスは以下。

- `<火>`: d4
- `<地>`: d6
- `<風>`: d8
- `<水>`: d20
- `<闇>`: d10
- `<光>`: d12

## 2. 出目の扱い

- 出目は**循環**する。最大値の次は1、1の前は最大値として扱う
- 各ダイスは **未反応 / 反応済み** の状態を持つ
- 未反応で反応条件を満たすダイスは **反応可能** である
- 反応効果で出目が変更されても、ダイスの状態は変化しない
- 反応効果の対象は、ダイスプール内から状態を問わず選べる

用語:

- **裏返す**: そのダイスの出目を反対側の目にする（N＋1−出目）
- **循環＋k / 循環−k**: 出目を環状に k 進める・戻す。最大値の次は 1、出目 1 の前は最大値
- **半回転**: 面数 N のダイスの出目を N÷2 だけ循環させて進める（循環＋N/2 と同じ）
- **振り直す**: そのダイスをもう一度振り、出目を置き換える
- **澱み**: 達成値の計算において、そのダイスを −1 として扱うこと

## 3. 判定の流れ

### 3-1. ダイス決定（ダイスプールの構築）

- ダイスプールにダイスを追加し、今回の判定で振るダイスを決定する
- 原則として、ダイスプールは **d10 を1つ追加した状態**から始める

※ サブシステムごとの「何を根拠に属性ダイスを何個足すか」は、各担当ファイルに記載する。

### 3-2. ダイスロール

- ダイスプール内のダイスをすべて振り、すべてを未反応にする

### 3-3. 反応

- 反応可能なダイスがある限り、以下の手順を繰り返す
  1. 反応可能なダイスを1つ選ぶ
  2. そのダイスの反応効果を適用する
  3. そのダイスを反応済みにする
  4. 新たに反応条件を満たした未反応ダイスがあれば、それも反応可能になる

### 3-4. 澱み

- 以下に該当するダイスに**澱み**が発生する
  - **水(d20)**: 出目が **3以下**
  - **それ以外**: 出目が **1**
- 澱みが発生したダイスは、次の達成値決定において **−1** として扱う

### 3-5. 達成値決定

- 各ダイスを次のように変換し、合計した値を達成値とする
  - 澱みのダイス：**−1**
  - 水(d20)（澱みでない場合）：⌊出目 ÷ 2⌋（端数切り捨て）
  - その他（澱みでない場合）：出目そのまま

- 目標値が指定されている場合、達成値が目標値以上なら成功、未満なら失敗となる

## 4. 属性ダイスごとの反応効果

### 火(d4)

- 反応条件: **3以上**
- 反応効果: **延焼**
  - 自身の出目を4にする
  - ダイスプールに d4 を1つ追加して振り、それを未反応として扱う

### 地(d6)

- 反応条件: **6**
- 反応効果: **変動**
  - 自身以外のダイスを1つ選び、出目を1〜2進めるか戻す

### 風(d8)

- 反応条件: **8**
- 反応効果: **旋回**
  - 自身以外のダイス 1 つを選び、そのダイスを半回転する

### 水(d20)

- 反応条件: **20**
- 反応効果: **流転**
  - 自身以外のダイス1つを選び、それを振り直す

### 闇(d10)

- 反応条件: **10**
- 反応効果: **覚醒**
  - 自身以外のダイス1つを選び、それを裏返す

### 光(d12)

- 反応条件: **12**
- 反応効果: **昇華**
  - 自身以外のダイス1つを選び、出目を最大値にする

## 5. 難易度帯の目安

現時点では以下を採用する。

- 5〜6: 易しい
- 7〜8: 普通
- 9〜10: 難しい
- 11〜12: 困難
- 13〜14: 非常に困難
- 15〜16: 無謀
- 17以上: 不可能級

この表は、1d10 と基本的な属性ダイス1個前後を前提にした暫定目安である。
