# 想成のスコーリア ルールブック

最終更新: 2026-06-24  
位置づけ: **プレイヤー（PL）向けに公開するルールの正本**。

## このサイトについて

このサイトは [`docs/`](https://github.com/yonpachi/ScholiaOfEidopoiesis/tree/main/docs) に置かれた Markdown をそのまま公開したものです。  
ルール本文は今後ここに整備していきます。

## 公開範囲

| 場所 | 内容 | 公開 |
| --- | --- | --- |
| `docs/`（この文書群） | PL 向けルール正本 | 本サイト |
| `catalog/` | ゲームデータ（印記・タグ・アイテム等） | カタログページ |
| `legacy/` | 開発用の旧文書・検討ログ | **非公開**（リポジトリ内参照用） |

## カタログ

印記・タグ・アイテム・武器・防具・スコーリオン・用語の一覧は、サイドバー「カタログ」から参照できます。  
データの正本は `catalog/game.sqlite` です。

## 更新について

ルール改訂は `docs/` を直接編集し、`main` ブランチへ push すると本サイトに自動反映されます。
