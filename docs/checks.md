## 基本ダイス

判定では、まず **d10** を使います。
必要に応じて、属性マナに対応する **属性ダイス** をプールに追加して振ります。

| 属性 | ダイス | 反応条件 | 反応効果                                    |
| ---- | ------ | -------- | ------------------------------------------- |
| 火   | d4     | 3以上    | **延焼** — 自身を4にし、d4を1個追加して振る |
| 土   | d6     | 6        | **変動** — 他ダイス1個を循環±1または±2      |
| 風   | d8     | 8        | **旋回** — 他ダイス1個を半回転              |
| 水   | d20    | 20       | **流転** — 他ダイス1個を振り直す            |
| 闇   | d10    | 10       | **覚醒** — 他ダイス1個を裏返す              |
| 光   | d12    | 12       | **昇華** — 他ダイス1個を最大値にする        |

属性は、タグや印記の表記（`<火>` `<土>` など）と対応します。

ダイスプールは、原則 **d10 を1個足した状態から** 構築します。

---

## 用語

- **循環** — 出目は環状に数えます。最大値の次は1、1の前は最大値です
- **未反応 / 反応済み** — 各ダイスはどちらか一方の状態を持ちます
- **反応可能** — 未反応で、反応条件を満たしているダイスです
- **循環±k** — 出目を k 進める、または k 戻します
- **半回転** — 面数 N のダイスを、循環＋(N÷2) だけ進めます
- **裏返す** — 出目を反対側にします（N＋1−出目）。d4 なら 1↔4、2↔3 です
- **振り直す** — そのダイスをもう一度振り、出目を置き換えます
- **澱み** — 達成値の計算で、そのダイスを **−1** として扱うことです

反応効果で出目が変わっても、ダイスの **未反応 / 反応済み** の状態は変わりません。
効果の対象を選ぶとき、特に指定がなければ、未反応・反応済みを問わず選べます。

## 判定の流れ

### 1. ダイス決定

ダイスプールにダイスを追加し、今回の判定で振るダイスを決めます。

### 2. ダイスロール

プール内のダイスをすべて振り、すべて **未反応** にします。

### 3. 反応

反応可能なダイスがある限り、次を繰り返します。

1. 反応可能なダイスを1つ選ぶ
2. そのダイスの反応効果を適用する
3. そのダイスを **反応済み** にする
4. 新たに反応条件を満たした未反応ダイスがあれば、反応可能になる

### 4. 澱み

次のダイスに **澱み** が発生します。

| ダイス    | 澱みになる出目 |
| --------- | -------------- |
| 水（d20） | **3以下**      |
| それ以外  | **1**          |

### 5. 達成値決定

各ダイスを次のように換算し、合計を **達成値** とします。

| ダイス               | 達成値への換算             |
| -------------------- | -------------------------- |
| 澱み                 | **−1**                     |
| 水 d20（澱みでない） | ⌊出目 ÷ 2⌋（端数切り捨て） |
| その他（澱みでない） | 出目そのまま               |

目標値が指定されているとき、**達成値 ≧ 目標値** なら成功、未満なら失敗です。

## 属性ダイスと反応効果

### 火（d4）— 延焼

- **反応条件**: 3以上
- **反応効果**: 自身の出目を **4** にする。ダイスプールに **d4 を1個追加** して振り、それを未反応として扱う

### 土（d6）— 変動

- **反応条件**: 6
- **反応効果**: 自身以外のダイス1個を選び、出目を **循環±1 または ±2** する

### 風（d8）— 旋回

- **反応条件**: 8
- **反応効果**: 自身以外のダイス1個を選び、**半回転** する

### 水（d20）— 流転

- **反応条件**: 20
- **反応効果**: 自身以外のダイス1個を選び、**振り直す**

### 闇（d10）— 覚醒

- **反応条件**: 10
- **反応効果**: 自身以外のダイス1個を選び、**裏返す**

### 光（d12）— 昇華

- **反応条件**: 12
- **反応効果**: 自身以外のダイス1個を選び、出目を **最大値** にする

## 例

**プール**: d10＝6、d6＝6、d8＝3（目標値 10）

1. **ロール** — すべて未反応
2. **反応** — d6 が6なので反応可能。**変動** で d10 を循環＋2 → d10＝8（6→8 が最も利得が大きい）。d6 を反応済みにする
3. **反応終了** — d8 は条件未達
4. **澱み** — 該当なし
5. **達成値** — d10＝8、d6＝6、d8＝3 → 合計 **17** → 成功
