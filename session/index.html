    セッションの流れ

    セッションと一日¶

    本作では、1回のTRPGセッション ＝ ゲーム内の1日 として進行します。

    • 1日は 朝 → 昼 → 夕 → 夜 の4フェイズで進みます
    • 夜フェイズが終わった時点で、その日＝そのセッションは終了 します
    • 次回のセッションは、翌日の朝フェイズ から再開します
    • 夜が終わったあと、続けて次の日に進むかどうかを、参加者で決めて構いません

    1日ごとに準備を整え、探索に出て、素材を持ち帰り、アイテムを作り、納品していくのが基本的な流れです。
    これを繰り返し、依頼・工房・研究・地景の開放が少しずつ前に進んでいきます。

    朝と夜の処理は、下表の順に進めます。

    一日の概要¶

    フェイズ 主な内容
    朝 日付進行・回復・依頼・準備
    昼 探索 または 街務（2回まで）
    夕 昼と同じ（夕の探索は危険度+2）
    夜 納品・成果の確定 → セッション終了

    朝フェイズ¶

    朝は、その日の準備や開始処理を行う時間です。以下の順に処理します。

    1. 日付が進む — 新しい一日が始まります。曜日の変化で影響を受ける要素があれば、この時点で更新します
    2. HPの回復 — 各PCはHPを最大値まで回復します
    3. 評判の自然低下 — 評判は毎朝、少しずつ低下します。何もしなくても街からの信頼は薄れていくため、依頼達成などで維持していく必要があります
    4. 依頼の期限日数の減少 — 期限付き依頼の残り日数が減ります。残り0日になった依頼は、その日の夜が最後の納品機会 です
    5. 依頼の追加 — その日に受けられる新しい副依頼を追加します。主依頼が新しく提示される場合もこのタイミングです。副依頼は、日々の小さな目標や収入源、評判を得るための手段になります
    6. 依頼の受注 — 現在受けられる依頼の受注を行います
    7. スコーリオンの配置 — スコーリオンを手に入れている場合、界標として据えることができます。マップ上の霧域に配置し、そこを地景として訪れられるようにします
    8. 施設の設置 — 研究が完了した施設などは、素材やお金を払って設置できます

    HP回復の例外¶

    全滅から帰還したPC は HP1 で帰還しており、その日の朝フェイズではHPは回復しません。

    昼フェイズ¶

    昼は自由に行動できる時間です。

    まず、霧域や地景へ 探索 に出るか、街の中で 街務 を行うか、どちらか1つだけ を選びます。
    PC全員が同じ選択をする必要はなく、別行動も可能です。

    探索¶

    探索を選んだPC全員で、霧域か地景1つを選び、その場所の探索を行います。
    トランプを使ってイベントを進め、素材や成果を持ち帰ります。

    • 探索中は、街にいる人や倉庫とのやり取りは原則できません
    • この時間の終了時、探索を行ったPCは街に帰還します
    • 持ち帰った素材は倉庫へ入りますが、そのフェイズ中に追加で街の行動はできません

    詳細

    探索の手順は別ページで掲載予定です。

    街務¶

    街にいる間は、相談や演出、確認、倉庫とのやり取りなどは自由に行えます。
    ただし、ルール上の効果を持つ行動 は街務回数を消費します。

    街務を選んだPCは、以下から 2回まで 選んで行えます（同じ行動を2回選んでもかまいません）。

    1. 錬金 — 作成できるアイテムから1つを選び、錬金を行う
    2. 買い物 — この街にあるお店から買い物を行う
    3. 交流 — 街の住人や街にいる他PCと交流を深める
    4. 調査・研究 — 依頼のための調査、または街・工房を発展させる研究を行う

    夕フェイズ¶

    夕も昼と同様、探索 か 街務 のどちらか1つを選んで行います。
    街務を選んだPCは、昼と同じく2回まで行動できます。街務の内容も昼と同じです。

    ただし 夕に探索を行う場合、その探索には危険度+2の補正 がかかります。

    夜フェイズ¶

    夜は、その日の成果を確定する時間です。以下の順に処理し、1日を終えます。

    1. 依頼の達成 — 受注している主依頼・副依頼の達成条件を満たしていれば、納品・確認を行います。期限0日の依頼を達成できなかった場合、ここで失敗となります
    2. 評判の反映 — 依頼などの成否に応じて、評判を増減します
    3. 研究成果の確定 — その日に進めた研究があれば成果を確定します。研究が完了した場合、翌日の朝 から施設を設置できます
    4. その日の成果の確定 — 得た素材や倉庫の状況などを確認し、1日を終えます

    昼・夕の共通ルール¶

    探索を選んだ場合¶

    • 探索は 1フェイズ分を丸ごと消費 します
    • 昼・夕の開始時に、そのフェイズで探索に出るか決めます
    • 探索に出ると決めた時点から、そのフェイズ中は街務や倉庫とのやり取りができません
    • 探索から帰還した後も、同じフェイズ中に街の行動はできません

    アイテムを使えるタイミング¶

    場面 タイミング
    町中 いつでも
    探索中 山札からカード2枚を公開する前まで
    戦闘中 各自の手番に1回（詳細は戦闘ルール掲載予定）

    一日を通じたルール¶

    HPと帰還¶

    • 探索中にHP0以下になって帰還したPCは、町へ HP1で帰還 します
    • 全滅時の詳しい処理は、探索・戦闘の各ルールで説明します

    マナ¶

    • マナは 日をまたいで完全に持ち越し ます

    バフ・デバフ¶

    • 基本的に 1戦闘中 または 1フェイズ中 のみ持続します
    • それより長く続く効果は、個別の効果文に従います

    倉庫と素材¶

    • 探索中以外は、倉庫との交換が可能です
    • 探索から帰還した瞬間、持ち帰った素材は倉庫へ入ります
    • 手持ちのアイテムは、そのまま保持して構いません

    全滅してもその日は続く¶

    • 昼フェイズに全滅した場合でも、夕フェイズ へ進みます
    • 夕フェイズに全滅した場合でも、夜フェイズ へ進みます
    • 依頼の期限や日程の進行は、通常どおり処理します
