## 0. 前提: ダイス体系

### 属性とダイス対応

| 属性 | ダイス |
| ---- | ------ |
| 火   | d4     |
| 地   | d6     |
| 風   | d8     |
| 闇   | d10    |
| 光   | d12    |
| 水   | d20    |

基本判定は **d10（闇）を1個** を起点に、属性ダイスを加えてプールを組む。

---

### 出目の読み方（達成値換算）

| ダイス           | 出目      | 達成値         |
| ---------------- | --------- | -------------- |
| d4/d6/d8/d10/d12 | 1         | **−1**         |
| d4/d6/d8/d10/d12 | 2〜最大値 | 出目そのまま   |
| d20              | 1〜3      | **−1**         |
| d20              | 4〜20     | **⌊出目 ÷ 2⌋** |

---

### 出目操作の用語

| 用語                   | 計算                | 意味                                                     |
| ---------------------- | ------------------- | -------------------------------------------------------- |
| **裏返す** (invert)    | N+1−出目            | 出目を反対側の目にする                                   |
| **循環+k** (cyclic)    | (出目−1+k) mod N +1 | 出目を環状に +k 移動                                     |
| **循環−k** (cyclic)    | (出目−1−k) mod N +1 | 出目を環状に −k 移動（出目2 → 最大値、出目1 → 最大値−1） |
| **半回転** (half_turn) | cyclic +N/2         | 偶数面ダイスのみ                                         |
| **振り直し**           | 1..N の新たな乱数   | 出目を置き換える                                         |

## 1. 反応システム

### 反応トリガー条件

| ダイス   | 発動条件  |
| -------- | --------- |
| d4 (火)  | 出目 ≥ 3  |
| d6 (地)  | 出目 = 6  |
| d8 (風)  | 出目 = 8  |
| d10 (闇) | 出目 = 10 |
| d12 (光) | 出目 = 12 |
| d20 (水) | 出目 = 20 |

### 反応処理の流れ

1. ロール後、トリガー条件を満たす未反応ダイスをすべてキューに積む
2. キューから先入れ順に取り出して効果を発動、そのダイスを **反応済み** にする
3. 効果の結果、新たにトリガー条件を満たしたダイスがあればキューに追加する（連鎖）
4. キューが空になるまで繰り返す

---

### 各属性の反応効果

#### 火 (d4)：自己強化 ＋ 増殖

- 自身の出目を **4（最大値）に固定**
- **d4 を 1個追加ロール**（連鎖可）

#### 地 (d6)：微調整

- 自身以外・未反応の 1個を **循環 ±1 または ±2** の4方向から利得最大の方向へ調整
- 調整後にトリガー条件を満たせば連鎖

#### 風 (d8)：中間操作（モード切替）

- **`WIND_HALFTURN` モード**（現在設定）  
  自身以外・**偶数面ダイス**の未反応 1 個を **半回転**（+N/2 循環）
- **`WIND_INVERT` モード**（代替）  
  自身以外・未反応の 1 個を **裏返す**
- 利得 > 0 のときのみ発動。連鎖なし。

#### 闇 (d10)：反転

- 自身以外・未反応の 1 個を **裏返す**（N+1−出目）
- 利得 > 0 のときのみ発動。連鎖可。

#### 光 (d12)：最大値固定

- 自身以外・未反応の 1 個を **最大値に固定**
- 利得 > 0 のときのみ発動。連鎖可。

#### 水 (d20)：振り直し

- 自身以外・未反応の 1 個を **振り直し**（結果は必ず受け入れ）
- 期待利得 > 0 の対象がなければ発動しない。連鎖可。

## 2. 種族能力

### 全体構造

**種族能力は「反応ループ内に1回差し込める追加アクション」である**。

- ロール後、通常の反応ループと同じタイミング帯の中で任意のタイミングに1回だけ発動できる
- 能力適用後も未反応ダイスが残っていれば反応ループは続行する
- シミュレーション上は「ロール直後（全ダイス未反応）に適用」として計測する（計算簡略化）
- **機械人のみ例外**：ロール前にプールを変更するため別案扱い

---

### 人間：裏返し（コスト = ダイス数）

- **対象**: 任意ダイス 1 つ（未反応のみ）
- **効果**: 出目を **裏返す**（N+1−出目）
- **連鎖**: 反転後のダイスは**反応済み（used=1）**として扱うため、反転ダイス自身は連鎖しない。他の未反応ダイスは引き続き通常通り反応する
- **コスト**: プール内のダイス数 を達成値から差し引く
- **使用判定**: 反転後に反応ループを走らせた後の達成値合計がベースライン（能力不使用）を上回る場合のみ使用。ダイスの即時利得 ≤ 0 の場合は候補から除外。

> コストがプール規模に比例して増大するため、ダイス数が多いほど発動ハードルが上がる。小プールでも安定した恩恵を得やすくなった。

---

### 機械人：d10 → d12 換装

- **タイミング**: 判定**前**（ダイスロール前に適用）
- **効果**: プール内の d10 を 1 個 **d12（光）に置換**してロールする
- **全ダイスの -1 ルール適用なし**：機械人の判定中は -1 ルールが発動しない。出目は変換式そのままで読む
  - d4/d6/d8/d10/d12: 出目 1 → **1**、出目 2〜最大値 → 出目そのまま
  - d20: 出目 1 → **0**、出目 2〜3 → **1**、出目 4〜20 → ⌊出目 ÷ 2⌋
- **反応は通常通り発生する**：d12 を含む全ダイスが通常のトリガールールに従って反応・連鎖する
- **別試行**：プール構成自体が変わるため、他種族と別の独立試行で基準値と比較する
- **シミュレーション補正**: **なし**（補正値 0）

---

### 獣人：ペナルティ付き振り直し

- **対象**: 全ダイス（反応済みを含む）から**期待利得最大**の 1 つ
- **効果**: 対象をペナルティ付き振り直し
  - 新出目が旧出目 **未満** → 出目を **1（= −1）に強制**
  - 新出目 ≥ 旧出目 → 新出目をそのまま採用
- **連鎖**: 振り直し後は **used=0**（未反応）→ 出目がトリガー条件を満たせば連鎖発火
- **シミュレーション補正**: なし（補正値 0）

> 旧出目が低いダイスほど期待利得が高いが、下振れしたときのペナルティリスクも上がる。成功時は連鎖も期待できる。

---

### ホムンクルス：三択

ロール・反応ループ完了後に、以下の3択から**最も達成値が高くなる1択**を自動選択して適用する。

#### 択 A：「エラー修正」（コスト −1）

- **タイミング**: 反応ループ完了後の最終状態に適用
- **効果**: 全ダイスの出目をペナルティなしで集計する
  - d4/d6/d8/d10/d12: 出目 1 → **1**（通常 −1 のところを 1 として読む）、出目 2〜最大値 → 出目そのまま
  - d20: 出目 1〜3 → ペナルティなしで読む
- **コスト**: 達成値 −1
- **使い所**: −1 のダイスが複数出た「外れ判定」で有効

> −1 を強制される出目が多い局面ほど恩恵が大きい。コスト −1 があるため、−1 のダイスが 0〜1 個だけでは採用されにくい。

#### 択 B：「連鎖報酬」（コストなし）

- **タイミング**: 反応ループ完了後の最終状態に適用
- **効果**: 反応済みダイスの枚数 ÷ 2（切り捨て）を達成値に加算する
  - 例: 反応 4 回 → +2、反応 3 回 → +1、反応 1 回 → +0
- **コスト**: なし
- **使い所**: 連鎖がよく起きた「当たり判定」で有効。反応が 0〜1 回では効果なし

> 反応が多い判定ほど上乗せが増える「連鎖ボーナス」。択 A（外れ向け）と相補的な関係。

#### 択 C：「ノンセンス変異」（コストなし）

- **タイミング**: ロール直後（raw 状態に適用）、その後 react_loop を走らせる
- **効果**: raw 状態で die_value が最も高いダイス 1 枚を**除外**し、残り全てを**循環 +1** → react_loop
- **コスト**: なし（高出目ダイス 1 枚の消失が自然なコスト）
- **使い所**: プール全体を一段階シフトして連鎖を誘発したい時（n ≥ 2 必要）

> 最も良いダイスを犠牲に残りを底上げする「ノンセンス変異」。犠牲が大きいほど残りへの恩恵も大きくなる。

> 択 A は −1 が多い外れ判定向け、択 B は連鎖が多く起きた当たり判定向け、択 C は全体シフトで連鎖を狙う場面向け。

---

### 付喪：下限保証振り直し

- **タイミング**: 反応ループ内（ロール直後に適用）
- **対象**: 全ダイス（反応済み含む）から**期待利得最大**の 1 つ
- **効果**: 対象ダイスを振り直す
  - 新出目 ≤ 旧出目 → **旧出目を維持**（下振れなし）
  - 新出目 > 旧出目 → **新出目を採用**
- **振り直し後のダイスは必ず反応済み（used=1）になる**（連鎖発火なし）
- 他の未反応ダイスは引き続き通常通り反応する
- 期待利得 > 0 の対象があるときのみ使用
- **シミュレーション補正**: **+1**

> 獣人（下振れで −1 強制 / 連鎖発火あり）との比較：  
> 付喪B は「下振れなし・振り直し後は反応済み」で安全性が高い一方、獣人は連鎖の爆発力を持つ。

---

### 半霊：反応済みリセット

- **タイミング**: 通常反応ループ**完了後**
- **対象**: `used=1`（反応済み）かつトリガー条件を満たすダイスの中から、**再発火時の期待利得が最大**の 1 つ
- **効果**: 対象ダイスを `used=0`（未反応）に戻し、再度 run_react_loop を実行する
- **不発条件**: 反応済みかつトリガー可能なダイスが存在しない場合は何もしない
- **シミュレーション補正**: なし（補正値 0）

> 人と霊の間の存在として、「一度終わった反応の境界を取り払う」効果。反応ループが終わった後に1つのダイスの境界を消し去り、連鎖を再起動させる。反応が多く起きた判定ほど候補が増えて強くなる。

## 3. 強さ設計の現状メモ

| 種族         | コスト／補正                                              | 主な強み                                                                               |
| ------------ | --------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| 人間         | コスト = ダイス数                                         | 低出目→高出目への変換。反転ダイスは反応済みのため連鎖しない。小プールほど有利          |
| 機械人       | シミュレーション補正なし                                  | -1ルール発動なし（出目1も正値）。反応は通常通り発生。d12置換の安定感                   |
| 獣人         | シミュレーション補正なし                                  | 高い期待利得。下振れで−1確定のギャンブル性。成功時は連鎖も期待できる                   |
| ホムンクルス | 択A: コスト−1 / 択B: コストなし / 択C: コストなし（除外） | 状況適応型。択Aは外れ判定の−1修正、択Bは連鎖回数×½加算、択Cは最高ダイス除外→全体循環+1 |
| 付喪B        | シミュレーション補正 **+1**                               | 純粋な安全網。下振れなし。振り直し後は連鎖なし。獣人との対比がキャラ                   |
| 半霊         | シミュレーション補正なし                                  | 反応後にトリガー済みダイスを再発火。反応が多い判定ほど強い。境界越えの連鎖再起動       |

### バランス所見（シミュレーション結果より）

- **ダイス総数 5〜7 付近**：全種族がほぼ同水準に収束しており、バランスが良好
- **ダイス総数 8 以上**：ホムンクルスと半霊が他種族より強めに伸びる傾向あり
- 通常ラインを「5個前後」として設計すれば、現在の強さバランスは許容範囲内と判断できる

## 4. 数値設計の前提（判定ダイス総数の目標感）

今後のすべての数値設計は、以下のダイス総数の目安を前提として行う。

| ライン       | ダイス総数  | 位置づけ                 |
| ------------ | ----------- | ------------------------ |
| 普段使い     | **5個前後** | 日常的な判定の標準ライン |
| 頑張るライン | **6〜7個**  | 少し無理をする局面       |
| 本気ライン   | **8個以上** | いざという時の全力投入   |

基本構成のイメージ：**1d10（固定）+ 精霊による無料分 + 追加 1〜2 個** が普段使いの標準。

### この前提が種族能力バランスに与える影響

- 種族能力の強さはダイス総数に依存して変化するため、「どのダイス数が通常ラインか」を先に固定することがバランス評価の前提となる
- 8個以上が頻発する設計にすると、ホムンクルス・半霊が相対的に強く見えやすくなる
- **高プールは珍しい**こと自体を、ルールとコスト設計で担保することが重要

---

## 5. 今後の設計課題

以下を「普段使い5個前後」の前提に合わせて決める必要がある。

- 精霊の無料提供ダイス数
- 追加ダイスのコスト（資源消費量）
- スキル・アイテムで増やせるダイスの上限・コスト
- 6〜7個に届くための消費量の妥当な重さ
- 8個以上が「本気の一手」として機能するコスト設定

**種族能力の最終バランスは能力単体で完全に揃えるのではなく、通常ラインのダイス数を先に固定した上で評価する。**
