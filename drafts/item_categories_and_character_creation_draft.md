# アイテムカテゴリ・キャラ作成ルール 草案

> 作業仮決定版（2026-04-11）  
> 正本化前の整理用ドキュメント。未確定項目は末尾に記載。

---

## 前提

- 本作の**基本単位はアイテム**。タグはアイテムに付属し、タグ抽出は基本ルール外。
- クラス設計は固定しない。「何屋か」は町での顔であり、機械的クラス名と一致させる必要はない。
- 戦闘は**タイムライン方式**。カードの数字は待機値として扱い、ダメージだけでなく**行動順操作**が重要。

---

## 1. キャラクター作成の考え方

- **職業名を機械的クラスにしない**方向が有力。
- 代わりに、**扱えるアイテム種 / レシピ系統を2つ選ぶ**のが実質のクラス選択（正式ルール）。
- 「何屋か」は自由設定。町での顔・商い・雰囲気を表す。
- 成長で扱える系統を増やすことができる（工房の成長に連動）。
- 未習得系統：**レシピを使用不可**（一旦完全不可とする）。

### テストプレイ最小構成（4クラス固定）

テストプレイ段階では、系統を自由選択せず以下の4クラスから1つを選ぶ（系統は**1種のみ**）。
装備製作・継続効果・特殊メカニズム（銃・ポーション等）は含めない。
正式版では現在候補に上がっているアイテム種も追加予定。

| クラス        | 系統 | 担当           |
| ------------- | ---- | -------------- |
| 火力          | 爆弾 | 単発ダメージ   |
| 回復          | 薬   | 単発回復・治療 |
| バフ          | 香料 | 単発強化       |
| デバフ/スタン | 札   | 単発妨害・封印 |

> テストプレイ終了後、正式ルール（系統2種を自由選択）に移行する。

### 専門性の分離

| 要素         | 内容                                      |
| ------------ | ----------------------------------------- |
| 町での顔     | 自由設定（何を営んでいるか）              |
| 機械的専門性 | 扱えるアイテム種 / レシピ系統の組み合わせ |

## 3. 消耗品カテゴリ（仮採用 12種）

### カテゴリ一覧と担当イメージ

| カテゴリ   | 担当                                                                                   |
| ---------- | -------------------------------------------------------------------------------------- |
| 薬         | 単発・単体の回復 / 治療。安定した直接支援                                              |
| 香料       | 単発・単体の強化・バフ。薬とは異なる手段（嗅覚・気分・身体の底上げ）で能力を引き上げる |
| ポーション | タグ依存が強い特殊カテゴリ。自由付与枠多め、副次効果混入あり。ベース弱め・上振れ強め   |
| 料理       | 戦闘外・全体寄りの回復 / 治療 / 強化。素材を砕かず価値化するルート                     |
| 爆弾       | 単発ダメージ担当。デバフは副次効果まで。継続ダメージは兵器へ                           |
| 呪具       | 継続デバフ担当。ダメージは主担当にしない                                               |
| 弾薬       | 銃必須・単発・単体寄りの万能系。回復 / 強化 / 弱体 / ダメージを担えるが器用貧乏寄り    |
| 兵器       | 継続ダメージ担当。設置・残留・自動攻撃寄り                                             |
| 常備具     | 探索・イベント用。戦闘中は基本使わない                                                 |
| 加護       | 継続回復 / 継続バフ担当。場や状態として持続する支援                                    |
| 札         | 単発デバフ / 封印 / 解除 / 妨害。爆弾のデバフ版に近い                                  |
| 触媒       | 単独完結しない副資材。他カテゴリの範囲・安定性・持続・事故率を補助                     |

### 料理の方向感

| 種別   | 効果方向 |
| ------ | -------- |
| ご飯   | 回復     |
| 飲み物 | 治療     |
| お菓子 | 強化     |

### 薬 / 香料 / ポーションの対象ルール

- 自分以外へ適用するには、**スロワー系補助具が必要**。
- 遠隔で味方・敵を対象にするには補助具が必要。
- 目的：薬 / 香料 / ポーションが爆弾・札の役割を食いにくくする。

### 弾薬の実体 / 想成の扱い

- **薬莢だけ実体で持ち込む**。
- 中身の火薬・特殊効果は**想成で込める**。
- 実体側は準備が必要 / 想成側はその場対応が可能。
- 完全な万能即応ではなく、薬莢という下地は必須。
- **弾薬が効果を決め、銃が射撃形態や対象数を決める**。

## 4. 装備カテゴリ（仮採用 6種）

### カテゴリ一覧

| カテゴリ   | 備考                                             |
| ---------- | ------------------------------------------------ |
| 武器       | 近接。3系統 × 3重量の9種構成                     |
| 盾         | 打ち直し / 更新対応                              |
| 銃         | 独立カテゴリ。パーツ差し替え更新型               |
| 防具       | 重 / 軽の2区分                                   |
| アクセサリ | 詳細未整理                                       |
| 補助具     | スロワー等の特殊使用器具。手装備への取り付け方向 |

### 装備スロット

| スロット   | 数  |
| ---------- | --- |
| 手         | 2   |
| 胴体       | 1   |
| アクセサリ | 2   |
| 補助具     | 1   |

### 各装備の仮方針

**銃**

- 複数パーツ構成の更新型装備。本体ごと作り直すより**パーツ差し替えで更新**する方向。

**武器・盾**

- 新規作成と**打ち直し / 更新**を分ける。更新は新規より軽コスト。
- 武器・盾にもアタッチメントを許容。

**防具**

| 種別   | 防御値 | 付与効果枠 | イメージ   |
| ------ | ------ | ---------- | ---------- |
| 重防具 | 高め   | 少なめ     | 鍛冶系     |
| 軽防具 | 低め   | 多め       | 裁縫・布系 |

**補助具**

- スロワー等の特殊使用器具を担当。
- **単独で片手を埋めるより、手装備へ取り付ける**方向が有力。
- 不要なキャラはその枠を別の強化パーツに回せる。

## 5. 戦闘ルール（仮決め）

### ラウンド

- **1ラウンド = 12カウント**（秒数は厳密に扱わない）

### 行動段階の共通待機値

| 行動段階 | 待機値 |
| -------- | ------ |
| 軽行動   | 3      |
| 中行動   | 4      |
| 重行動   | 5      |
| 特行動   | 6      |

### 重量補正

| 重量     | 補正        |
| -------- | ----------- |
| 軽量武器 | 全待機値 -1 |
| 標準武器 | なし        |
| 重量武器 | 全待機値 +1 |

※重量補正だけで武器個性を作るのではなく、**武器種ごとの得意段階ボーナス**も別途設ける。

## 6. 近接武器 9種（3系統 × 3重量）

### 構成

| 系統         | 軽量 | 標準     | 重量     |
| ------------ | ---- | -------- | -------- |
| ダメージ     | 短剣 | 剣       | 大剣     |
| ノックバック | 拳   | フレイル | ハンマー |
| デバフ       | 槍   | 刀       | 斧       |

### 各武器の役割イメージ

| 武器     | 重量 | 手   | 役割イメージ                                       |
| -------- | ---- | ---- | -------------------------------------------------- |
| 短剣     | 軽量 | 片手 | 低火力。付与重ね向き。両手短剣で追加挙動の余地あり |
| 剣       | 標準 | 片手 | 完全な基準武器                                     |
| 大剣     | 重量 | 両手 | 高火力。両手ダメージ武器の基準                     |
| 拳       | 軽量 | -    | 待機値軽め。スタン寄り                             |
| フレイル | 標準 | 片手 | 崩し・ノックバック・妨害寄り                       |
| ハンマー | 重量 | 両手 | ノックバック大。スタン値上昇寄り                   |
| 槍       | 軽量 | -    | 牽制・迎撃・待機値妨害寄り                         |
| 刀       | 標準 | -    | 構え崩し。次の一手を通しやすくする弱体寄り         |
| 斧       | 重量 | 両手 | 防御値低下・装甲破壊寄り                           |

### 杖の扱い

- 9武器種には**含めない**方向が有力。
- **補助具寄りの特殊装備 / 媒体**として別枠扱い。
